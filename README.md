# Sample notebooks for IBM watsonx.ai for IBM Cloud and IBM watsonx.ai software product

Sample notebooks demonstrating Watson Machine Learning and wastsonx.ai capabilities like:
- running model building AutoAI or Deep Learning experiments
- deploying 3rd party models as web services or batch jobs (i.e.: scikit-learn, xgboost, keras, PMMl, SPSS etc.)
- monitoring of deployments with OpenScale (drift, bias detection)
- model lifecycle management (update model version, refresh deployment)
- inference watsonx.ai foundation models 
- integration LangChain with watsonx.ai 

Notebooks with python code and python SDK can be found under `python_sdk` folder. The REST API examples are organized under `rest_api` folder.
